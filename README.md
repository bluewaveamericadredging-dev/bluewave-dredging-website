# 🌊 BlueWave America Dredging Inc. Website

A modern, responsive corporate website for **BlueWave America Dredging Inc.**, a marine engineering and hydraulic dredging company specializing in large-scale infrastructure, environmental restoration, and coastal protection projects.

---

## 🚀 Overview

This website showcases BlueWave’s capabilities, projects, innovation, and company values through a high-end, interactive user experience optimized for both desktop and mobile devices.

Built with performance, scalability, and clean design in mind.

---

## 🧱 Features

- Fully responsive design (mobile + desktop optimized)
- Modern UI with smooth animations and transitions
- Interactive project & press popups (modal system)
- Scroll-based reveal animations
- Clean navigation with mobile menu
- High-performance static site (no backend required)
- SEO-friendly structure

---

## 🗂️ Pages Included

- **Home** – Company introduction and highlights  
- **About** – Company story, vision, and leadership  
- **Capabilities** – Core services and expertise  
- **Projects (Work)** – Featured projects with interactive modals  
- **Innovation** – Technology and forward-thinking solutions  
- **Safety** – Commitment to operational safety and standards  
- **Careers** – Opportunities and company culture  
- **Press** – News, announcements, and updates  
- **Contact** – Inquiry form (Formspree integration ready)

---

## 🛠️ Tech Stack

- HTML5  
- Tailwind CSS (CDN)  
- Vanilla JavaScript  
- Font Awesome (Icons)  
- Google Fonts (Inter + Playfair Display)
